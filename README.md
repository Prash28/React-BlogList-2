**Cloning the Repository**

```bash
git clone https://github.com/Prash28/CCBP-App-Store.git
cd dynamic-landing-page-1
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```


**Running the Project**

```bash
npm start
```
